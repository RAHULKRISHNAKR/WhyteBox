Open browser console (F12) and look for:

  • ❌ Any RED error messages

  • ✅ GREEN messages showing files loaded

Step 4: Instructions

  1. Make sure backend is running: python api/app.py in backend folder
  2. Make sure frontend is running: python -m http.server 8000 in frontend folder
  3. Hard refresh browser: Press Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  4. Open console: Press F12
  5. Click tests above and check results
